## F2 (分量重构): 重写当前失效或起反作用的奖励分量（改变其数学表达形式）

## F3 (平衡微调): 仅修改现有各分量的权重系数
适用场景：
- 各分量设计本身基本合理，但权重配比失调（如成功奖励远小于失败惩罚）
- 希望增加探索性（降低某些惩罚权重）或强化某类行为（提升某分量权重）
- 训练中出现某种行为但需要微调强度而非改变方向
- 只想保守调整，不想引入过多风险
